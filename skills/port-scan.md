# 端口扫描与资产探测

使用 fscan 对目标进行端口扫描和资产探测。

## 使用方式

```
fscan -h <target> [options]
```

## 常见场景

### 1. 单 IP 全端口扫描
```
fscan -h 192.168.1.1
```

### 2. 网段扫描
```
fscan -h 192.168.1.0/24
```

### 3. 指定端口
```
fscan -h 192.168.1.1 -p 80,443,3306,6379,8080
```

### 4. 仅存活探测（不爆破）
```
fscan -h 192.168.1.0/24 -nobr -nopoc
```

### 5. 从文件读取目标
```
fscan -hf targets.txt
```

### 6. 排除特定主机
```
fscan -h 192.168.1.0/24 -eh 192.168.1.1
```

### 7. 结果输出 JSON
```
fscan -h 192.168.1.0/24 -f json -o result.json
```

### 8. 静默模式 + 无结果保存
```
fscan -h 192.168.1.1 -silent -no
```

## 常用参数说明

| 参数 | 说明 |
|------|------|
| `-h` | 目标主机/IP段/域名 |
| `-hf` | 主机列表文件 |
| `-p` | 端口（默认1000常用端口）|
| `-pf` | 端口文件 |
| `-eh` | 排除主机 |
| `-nobr` | 禁用暴力破解 |
| `-nopoc` | 禁用POC扫描 |
| `-full` | 全量POC扫描 |
| `-f` | 输出格式: txt/json/csv |
| `-o` | 输出文件 |
| `-silent` | 静默模式 |
| `-t` | 端口扫描线程数（默认600）|
| `-time` | 端口超时秒数（默认3）|
| `-np` | 禁用ping探测 |
| `-m` | 扫描模式: all/icmp/指定插件 |
